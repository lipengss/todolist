推荐：两者结合

日期格 + 按钮（方案 A）

  • 日期格悬停时右上角显示小 +
  • 点击 + 或双击日期格 → 该格展开为内联输入框
  • 输入任务名后按 Enter 创建，日期自动填充为该格日期
  • 按 Esc 取消，恢复为正常日期格

右侧面板 + 按钮（方案 B）

  • 选中日期后，右侧面板标题栏始终显示 + 按钮
  • 点击后在面板顶部出现输入框
  • 输入 Enter 创建，自动关联当前选中日期
  • 同样 Esc 取消

涉及改动

  • CalendarView.tsx — 日期格添加悬停态和 + 按钮；右侧面板标题栏添加 + 按钮；管理内联输入状态
  • App.tsx — 暴露 handleAddTodo(date?: string) 方法，支持传入预填日期
  • 新任务默认值：当天日期 + 默认分类 + 中优先级
