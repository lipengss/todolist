交互细节

  • 验证码图片：白色背景 + 扭曲字符 + 干扰线，无任何文字覆盖
  • Hover 效果：边框高亮为紫色 + 光标变 pointer，提示可点击
  • 点击刷新：点击图片立即请求新验证码，带 500ms 防抖
  • 辅助提示：图片下方小字 "看不清？点击图片刷新"
  • 登录失败：验证码错误时自动刷新 + 清空输入 + 红色提示
  • 初始加载：页面加载时自动获取第一个验证码

后端 API

GET /api/auth/captcha → { svg, token }
svg: 验证码 SVG 字符串，token: 临时标识（5分钟过期，内存存储）

POST /api/auth/login + { ..., captchaToken, captchaText }
先验验证码，再验密码；校验后立即删除防止重用
